# Bug Reports for Hyperspin
Welcome to the official issue tracker for [Hyperspin](https://starpelly.itch.io/hyperspin)!

To submit issues and report bugs, you'll need a GitHub account. If you don't already have one, you can [create one for free here](https://github.com/signup).

## Submitting issues
**[Use this link to submit a new issue](https://github.com/Boxsubmus/hyperspin-issues/issues/new?template=bugreport.yml).**
This will automatically use the "bug report" template.
